Project Outline: Hashing algorithm is significantly important for computer memory systems. DHash is designed for asynchronous shared memory system with multiprocessors. The challenge for most existing hash tables is the lack of robustness since the pre-defined hash functions cannot be changed even if they cannot evenly distributed incoming data to the hash table buckets. This issue has affected in operating systems including Linux Kernel and programming languages such as Perl, PHP and .NET. DHash, a non-blocking dynamic hash table, stems from lightweight distributing mechanism that can atomically and efficiently distribute each node. DHash can consistently outperforms the alternatives of concurrent regular operations at different concurrency levels with different operation mixes and average load factors. Furthermore, the parallelized rebuild operation with eight or more threads outperforms the alternatives.
